# ILUC_NIPE Webapp

Single Page Application (SPA) for exploring land-use and land-cover (LULC)
change across the 133 Brazilian Intermediate Geographic Regions (RGINTs),
2008–2024. Built as a **static** site for GitHub Pages — no backend.

**Stack:** React 18 · TypeScript · Vite · Tailwind CSS · Recharts · Plotly.js ·
React Router (hash routing) · Papaparse.

## Views

| Route             | Description                                            |
| ----------------- | ------------------------------------------------------ |
| `/overview`       | National dashboard — KPIs, native-veg series, donut, Sankey |
| `/map`            | Interactive grayscale choropleth of the 133 RGINTs     |
| `/region/:id`     | Per-region analytical sheet (transitions, series, matrix, PAM) |
| `/transitions`    | National transition Sankey + top-20 table by period    |
| `/ranking`        | Sortable/searchable ranking of all 133 regions         |
| `/about`          | Methodology, sources, 15-class system, citation        |

## Develop

```bash
cd webapp
npm install
npm run dev          # http://localhost:5173/ILUC_NIPE/
npm run test         # unit tests (vitest)
npm run build        # type-check + bundle into ../docs
npm run preview      # serve the built /docs
npm run test:e2e     # Playwright smoke tests (needs `npx playwright install`)
```

## Deployment (GitHub Pages)

`npm run build` outputs to the repository-root `/docs` folder. Configure
**Settings → Pages → Source = Deploy from a branch**, branch = your default,
folder = `/docs`. The site base path is `/ILUC_NIPE/`.

## Data

All data is static under `public/data/` (copied into `/docs/data` on build)
and generated by the scripts in [`../scripts`](../scripts):

| File                              | Generator                          |
| --------------------------------- | ---------------------------------- |
| `rgint_meta.json`                 | `generate_rgint_meta.py`           |
| `rgint_indicators.csv`            | `generate_rgint_indicators.py`     |
| `national_timeseries.csv`         | `generate_rgint_indicators.py`     |
| `rgint_pam/{id}.csv`              | `generate_pam_by_rgint.py`         |
| `rgint_transitions/{id}.csv`      | `generate_rgint_transitions.py`    |
| `national_transitions.csv`        | `generate_national_transitions.py` |
| `rgint_simplified.geojson`        | `generate_geojson.py`              |

> **Note:** the full 15×15 transition matrices for all 133 RGINTs are not yet
> in the repository. The transition/matrix generators and the UI are fully
> data-driven: drop the complete per-region matrix JSONs into
> `webapp/data/rgint_matrix/`, re-run the generators, and the Sankey/heatmap
> views populate automatically — no code changes required.

Regenerate everything:

```bash
pip install pandas geopandas shapely openpyxl
cd scripts && for s in generate_*.py; do python "$s"; done
```
